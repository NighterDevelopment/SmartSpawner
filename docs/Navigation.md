## 🎨 How to Read This Wiki

### 📝 Notation Conventions

#### File Paths
```
plugins/SmartSpawner/config.yml
```

#### Commands
```bash
/ss reload
/ss give <player> <type> [amount]
```

#### Configuration Examples
```yaml
spawner_properties:
  default:
    min_mobs: 1
```

#### Code Examples
```java
SmartSpawnerAPI api = SmartSpawnerProvider.getAPI();
```
